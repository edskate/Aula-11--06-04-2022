Para que serve o debug?

Seu objetivo é encontrar erros que podem impedir que os códigos funcionem adequadamente.
Com ele, é possível determinar o que está ocorrendo dentro do código-fonte
e obter sugestões de ações de melhorias.
Cada instrumento do desenvolvimento
possui sua própria ferramenta para debugar um código.

Como usar o debug do Visual Studio?

Outra maneira de depurar uma instrução por vez é pressionar F11 .
O Visual Studio realça e exibe uma seta ao lado da próxima linha de execuçã Neste ponto, a janela locais mostra que a matriz está vazia name
currentDate e tem valores padrão.

Portanto, torna-se necessário o emprego de comandos para abrir o Arquivo HTML, quando necessário,
e para abrir a janela das ferramentas de desenvolvedor
Este recurso irá abrir em uma nova janela para utilização de dpuração e demias utilidades.
